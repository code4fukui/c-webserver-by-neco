# webserver in C language 

A web server implementation in C language with neco.

## Using

```sh
cc -c server.c neco.c
./a.out 8080
```
open http://localhost:8080/

## Dependencies

- [neco.h](neco.h) / [neco.c](neco.c) from [neco](https://github.com/tidwall/neco)
