# c-webserver-by-neco

C言語とコルーチンライブラリ [neco](https://github.com/tidwall/neco) を使用して実装された、シンプルで並行処理可能なWebサーバーです。利便性のため、`neco` のソースファイルも本リポジトリに同梱しています。

## 特徴

-   **並行処理を前提とした設計:** 受信した各接続を個別のコルーチンで処理します。
-   **ノンブロッキングI/O:** `neco` の効率的なノンブロッキングネットワーク関数を基盤としています。
-   **静的ファイルサーバー:** HTML、CSS、画像、その他のファイル形式を配信します。
-   **MIMEタイプの検出:** 一般的なファイル拡張子に対して、適切な `Content-Type` ヘッダーを自動的に設定します。

## 要件

-   Cコンパイラ（例: GCC、Clang）

## 使い方

サーバーをコンパイルして実行します。

```sh
# ソースファイルをコンパイル
cc server.c neco.c

# サーバーを実行し、ポート8080でリッスン
./a.out 8080
```

ブラウザで `http://localhost:8080/` を開き、デフォルトの `index.html` ページを表示します。
